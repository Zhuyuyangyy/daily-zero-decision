# PRIVACY POLICY

> 每日零决策卡（Daily Zero Decision / 养一片自己的天空）
> 仓库：`D:/ZYY Project/daily-zero-decision`
> App：`app/`
> 文档版本：v1.0.0（与应用 v0.1.0 配套）

我们把这一段写得很短，因为 **我们不收集任何东西**。

---

## 1. 一句话原则

> **你的数据只属于你。它只存在于你电脑上的浏览器里。我们看不到、拿不到、也从未拿过。**

---

## 2. 数据存放在哪里？

| 数据类型 | 存放位置 | 是否上传 | 加密 |
| --- | --- | --- | --- |
| 任务清单 / 今日卡 | 浏览器 `localStorage`（key: `daily-zero-decision`） | 否 | 否（明文 JSON） |
| 打卡记录（log / history） | 浏览器 `localStorage` | 否 | 否 |
| 连续天数（streak） | 浏览器 `localStorage` | 否 | 否 |
| 心情（moods） | 浏览器 `localStorage` | 否 | 否 |
| 成就 / 安心卡 / 宠物状态 | 浏览器 `localStorage` | 否 | 否 |
| 设置（每会话页数、上次读到哪等） | 浏览器 `localStorage` | 否 | 否 |
| 备份文件 | 由用户主动下载到本地 | 否 | 否 |

**清除浏览器站点数据 = 永久删除全部数据。** 我们没有"找回账号"功能，因为我们没有账号。

---

## 3. 我们**不**做的事

- **不**收集任何使用数据（点击、停留、页面访问）
- **不**上报任何崩溃 / 错误日志到远端
- **不**注入任何第三方脚本（Analytics / Ads / 社交分享 SDK）
- **不**使用 Cookie
- **不**使用 `IndexedDB` / `SessionStorage` / 浏览器指纹
- **不**要求账号注册、邮箱、手机号
- **不**发起任何出站网络请求（构建产物中无 `fetch` / `XMLHttpRequest` / `WebSocket` 调用）
- **不**读取你的通讯录、相册、麦克风、摄像头、位置、剪贴板

---

## 4. 我们**不**做但你**可以做**的事

因为应用是纯前端，你可以自由地：

- 在**离线**环境使用（飞行模式下打开已访问过的页面仍可工作）
- 在**本机**直接编辑导出后的 JSON 备份
- 用**任何**文本编辑器查看你的数据
- 把备份**加密**后存到你信任的网盘 / 邮箱 / U 盘
- 把**整站部署在你自己的服务器**（见 `vercel.json` / `README.md` 部署段）
- **删除** `localStorage` 中的 `daily-zero-decision` key 来彻底清除数据

---

## 5. 备份与迁移（Import / Export）

应用提供「导出 / 导入」两个按钮：

- **导出**：把 `localStorage` 中的完整状态序列化为 JSON，触发浏览器下载，文件名形如 `daily-cloud-backup-2026-06-23.json`。
- **导入**：从用户选择的本地 JSON 文件读取并校验，**完整覆盖**当前状态（导入前请自行导出当前状态）。

导入过程**不上传文件到任何服务器**，文件只在你的浏览器内解析。

数据结构与 `schemaVersion` 规则详见 `DATA_MODEL.md`。

---

## 6. 第三方依赖与隐私

- `react` / `react-dom` 是本应用仅有的**运行依赖**，来自 npm 官方 registry
- 这些库**不包含**任何默认的遥测或上报代码
- 开发依赖（Vite、TypeScript、Vitest、ESLint 等）仅在构建时使用，不会进入最终产物
- 完整依赖列表与锁定版本见 `app/package.json` / `app/package-lock.json`

我们**不会**在产品中动态加载任何外部资源（脚本、字体、样式、图片）。

---

## 7. 儿童隐私

本应用不收集任何个人信息，因此不区分成年 / 未成年用户。
若你是未成年人，请在监护人指导下使用；如你希望清除应用数据，按 §4 删除 `localStorage` 的 `daily-zero-decision` 即可。

---

## 8. 法规合规（自评）

- **GDPR**（欧盟通用数据保护条例）：因我们不收集任何个人数据，无 DPO 需求、无数据主体请求处理流程。
- **CCPA**（加州消费者隐私法）：同上，无适用条款。
- **中国《个人信息保护法》**：同上，不存在个人信息处理者义务。
- **COPPA**（美国儿童在线隐私保护法）：同上，不收集 13 岁以下儿童信息。

**注意**：本评估仅就"我们不收集数据"这一事实陈述。**你**如果在备份文件中加入了个人信息并自行上传到云端，相关合规责任在你；我们不为你的本地存储行为负责。

---

## 9. 政策变更

我们承诺：任何会让"数据被收集 / 被上传 / 被第三方看到"的变更，都必须：

1. 先在本文件显著位置标注
2. 同步在 `app/CHANGELOG.md` 的 `Breaking` 区段写明
3. 在仓库 README 顶部 banner 提示 ≥ 30 天

违反上述承诺的 PR 将不会被合并。
