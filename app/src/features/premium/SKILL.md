# 天空投资功能开发计划

## 功能概述

**「天空投资卡」**：付费购买保护，连续打卡中断后不会清零，仅消耗一张保护卡

## 核心特性

1. **保护卡系统**
   - 用户持有保护卡数量
   - 断签时自动消耗1张
   - 无保护卡时正常清零

2. **购买界面**
   - 简洁的购买弹窗
   - 显示当前保护卡数量
   - 购买后即时生效

3. **订阅模式**
   - ¥30/月 = 每月赠送2张保护卡
   - 自动续费

## 关键痛点

**用户为什么要买？**
- 防止努力白费的焦虑
- 给自己一个"安全网"
- 心理安全感

**为什么比订阅更好？**
- 买卡是即时满足（立即获得保护）
- 订阅是延迟满足（每月才给2张）
- 可以囤卡（一次性买10张）
