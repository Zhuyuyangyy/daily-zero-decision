# official-document-assistant v0.3 宣传视频配音稿

## 45 秒竖版口播稿

1. 很多材料不是不会写，而是写完之后没人能快速、稳定、按规范帮你初审。
2. 这个 Skill 的定位是人工复核前的公文初审助手，先检查、再修订、再提醒风险。
3. 它会检查标题、主送机关、落款、日期、附件说明和正文层级这些容易漏掉的细节。
4. 请示、报告、函、通知常常混用。v0.3 加了专项规则，能提示这些典型问题。
5. 只告诉你哪里错还不够。它会给出一版可复制、可人工复核的建议修订稿。
6. 会议纪要里谁负责、什么时候完成、做什么，它会整理成一张清单。
7. 政务材料不能只看写得好不好，还要注意手机号、身份证号、内部资料和涉密表述。
8. 它支持 .docx 输入和 batch 批量审查，适合先扫一批材料，再交给人工复核。
9. 作品已经发布到讯飞 SkillHub，搜索 official-document-assistant。热度榜看收藏量和下载量。

## 发布文案

基层政务、办公室、行政、学生组织写正式材料时，最耗时间的往往不是从零起草，而是写完后的第一轮规范审查、反复修改和会后督办整理。

我做了一个讯飞 SkillHub 参赛 Skill：official-document-assistant。它面向人工复核前的初审环节，支持公文格式审查、文种规则提示、建议修订稿生成、会议督办事项提取和敏感信息脱敏预览。
